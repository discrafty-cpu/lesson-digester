6
Vinculum Dark Core
glassmorphism · gradient text · dark navy · standard tags
DEFAULT VINCULUM THEME VINCULUM SCORE: 10/10
The direct translation of vinculum-core.css default theme to PPTX. Dark navy backgrounds (#0a0e1a), cyan→purple gradient titles, glassmorphism content cards with subtle borders, JetBrains Mono standard tags color-coded by framework (cyan=CCSS, purple=MN, pink=TEKS), and ambient glow orbs. This is the flagship — every other theme is a variation of this foundation.

CRA Flow

EXPLORE PRACTICE PLAY VINCULUM CRA FRAMEWORK
The most uniquely Vinculum layout — organized entirely around the CRA (Concrete-Representational-Abstract) framework that structures every Vinculum tool. Each lesson section is tagged with a mode: Explore (cyan) for concept introduction, Practice (green) for guided/independent work, Play (yellow) for application and gamification. The mode tabs at the top mirror the tool interface, creating a consistent experience between the lesson PPTX and the Vinculum digital tools students use alongside it.

Recommendation
Primary default: Style 6 (Vinculum Dark Core) — the flagship that matches every Vinculum tool. Teacher choice: Offer Style 8 (Arctic Frost) as the light-mode alternative for projector classrooms. CRA integration: Style 9 (CRA Flow) when lessons pair with Vinculum digital tools — it creates a seamless visual bridge between the presentation and the interactive practice. Keep from originals: Styles 1 and 5 serve important niches (formal observations and print notebooks) that the Vinculum aesthetic can't replace. Style 3 (Bold Modern) becomes redundant once Style 6 exists.
